# 下拉菜单（Dropdown）

## Overview

下拉菜单是显示多个选择的紧凑方式。它在与元素（如图标或按钮）交互或用户执行特定操作时出现，与操作入口紧密联系，出现在屏幕上其他内容的上方，通常位于触发元素的下方。可承载简单的筛选和选项功能。

**设计来源帧：** `下拉菜单`

---

## 组件构成

```
下拉菜单（Dropdown）
├── 下拉菜单（Dropdown）        用于整页筛选，与入口紧密联系
│   ├── 菜单栏（Menu Bar）      触发入口，横向排列筛选项
│   │   ├── 3列
│   │   └── 4列
│   └── 菜单列表（List Panel）  展开后的选项面板
│       ├── 单选（Single Select）
│       └── 多选（Multi Select）
└── 气泡卡片（Popover）         用于局部筛选，弹出浮窗形式
```

---

## 类型

| 分类 | 使用场景 | 关闭方式 |
|---|---|---|
| 下拉菜单（Dropdown） | 用于整个页面的筛选，与入口紧密联系，不可分割 | 点击菜单中的选项，或点击屏幕空白区域 |
| 气泡卡片（Popover） | 用于相对局部的筛选，由矩形 + 三角箭头组成的弹出浮窗，箭头指向触发控件 | 点击气泡卡片内的选项，或点击气泡卡片外的其他区域 |

## 一、菜单栏（Menu Bar）

菜单栏是下拉菜单的触发入口，常固定在页面顶部或模块顶部。各筛选项在菜单栏内均分列宽，文字与三角图标水平居中排列，标签与标签之间设置 16px 安全边距。

### 1.1 尺寸规范

| 属性 | 值 | Token |
|---|---|---|
| 高度 | 44px | — |
| 背景 | 白色 | `color-foreground-layer1` |
| 列宽（3列） | 各 125px | — |
| 列宽（4列） | 各 94px（末列 93px） | — |
| 列内左右内边距 | 各 8px | `padding-base` |
| 标签间安全边距 | 16px | `margin-loose` |
| 列间竖向分割线 | 1px | `color-divider` |

### 1.2 选项文字

| 属性 | 值 | Token |
|---|---|---|
| 字体 | PingFang SC Regular | `font-family-ios-cn` |
| 字号 | 14px | — |
| 颜色（默认） | `rgba(0,0,0,0.84)` | `color-text-primary` |
| 颜色（激活/已筛选） | `#2E58FF` | `color-brand-primary` |
| 对齐 | 列内水平居中 | — |

### 1.3 展开图标

| 属性 | 值 | 说明 |
|---|---|---|
| 图标 | `icon/功能图标/展开筛选` | 三角形 |
| 尺寸 | 8.33 × 6.25px | — |
| 收起状态 | 三角朝下 | — |
| 展开状态 | 三角朝上（Y 轴翻转） | — |
| 颜色（默认） | 同选项文字，`color-text-primary` | — |
| 颜色（激活） | `color-brand-primary` | — |

## 三、气泡卡片（Popover）

气泡卡片是由矩形内容区 + 三角箭头组成的弹出浮窗，箭头指向触发该气泡的控件或区域。适用于模块内局部筛选，选项数量相对较少时。

> **使用原则：**
> 1. 与入口紧密联系，不可分割；点击气泡内选项或屏幕其他区域可关闭
> 2. 只有当模块复杂、但选项相对简单时才使用
> 3. 一次只显示一个下拉菜单
> 4. 不建议选项过多或需要滚动操作；如需滚动，推荐改用 Bottom Sheet

### 3.1 容器规格

| 属性 | 值 | Token |
|---|---|---|
| 背景 | 白色 | `color-foreground-layer1` |
| 圆角 | 6px | `radius-large` |
| 最小宽度 | 88px | — |
| 宽度 | 随内容撑开，不超过屏幕宽度 | — |
| 蒙层 | **无**（不使用全屏遮罩） | — |

### 3.2 三角指示箭头

| 属性 | 值 |
|---|---|
| 尺寸 | 10 × 4px |
| 颜色 | 白色，与气泡背景一致 |
| 方向 | 指向触发该气泡的控件；触发控件在气泡上方时箭头朝上，以此类推 |

### 3.3 选项行

| 属性 | 值 | Token |
|---|---|---|
| 行高 | 44px | — |
| 文字字体 | PingFang SC Regular | `font-family-ios-cn` |
| 文字字号 | 14px | — |
| 文字颜色（默认） | `rgba(0,0,0,0.84)` | `color-text-primary` |
| 文字颜色（选中） | `#2E58FF` | `color-brand-primary` |
| 左右内边距 | 16px | `padding-extra-loose` |

### 3.4 子变体

气泡卡片选项行支持三种内容结构：

#### 纯文字

文字左对齐，左内边距 16px，无额外图形元素。

#### 带图例色点

| 属性 | 值 | 说明 |
|---|---|---|
| 色点尺寸 | 12 × 12px | 圆形，代表图例颜色 |
| 色点位置 | 距左边 16px，垂直居中 | — |
| 文字位置 | 距左边 30px（= 16 + 12 + 2px 间隙） | — |

#### 带操作图标

| 属性 | 值 | 说明 |
|---|---|---|
| 图标尺寸 | 约 17 × 17px | 功能图标 |
| 图标位置 | 距左边 16px，垂直居中 | — |
| 文字位置 | 距左边 40px（= 16 + 17 + 7px 间隙） | — |

## Constraints / Do & Don't

| | 规则 |
|---|---|
| ✅ | 菜单栏高度固定 44px，不随内容变化；菜单列表最大高度不得超过页面高度 80% |
| ✅ | 菜单栏文字使用 `font-family-ios-cn`（PingFang SC），不使用数字字体 |
| ✅ | 激活/已筛选状态的选项文字与图标均改为 `color-brand-primary` |
| ✅ | 展开菜单列表后，列表以下区域加 `color-background-mask-level3` 蒙层（`rgba(0,0,0,0.24)`） |
| ✅ | 单选状态用勾号图标（右侧），多选状态用圆形单选按钮（右侧） |
| ✅ | 气泡卡片箭头必须指向触发该气泡的控件或区域 |
| ✅ | 气泡卡片选项文字 14px，下拉菜单列表选项文字 16px；两种类型字号不可混用 |
| ✅ | 气泡卡片最小宽度 88px，不得小于此值 |
| ❌ | 不要省略蒙层，下拉菜单（Dropdown）展开时必须添加 `color-background-mask-level3` 遮罩；气泡卡片无需蒙层 |
| ❌ | 不要在菜单栏中混用不同字号或字重 |
| ❌ | 气泡卡片不可在整页筛选场景中代替下拉菜单使用 |
| ❌ | 气泡卡片选项过多需要滚动时，应改用 Bottom Sheet，不得在气泡内滚动 |
