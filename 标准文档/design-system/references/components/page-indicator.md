# PageIndicator 页面指示器

## 定义

页面指示器（也叫分页符，iOS 规范称 Page Control）用于指示当前页面在多页内容中的位置。

---

## 组件类型

按内容的翻动方式分为两类。

### 01 翻页指示器

**适用场景：** 轮播图、卡片横滑、引导页等——页面保持**整张翻动**并吸附到下一页。

| 属性 | 选中 | 未选中 |
|---|---|---|
| 宽度 | 12px | 6px |
| 高度 | 3px | 3px |
| 圆角 | 1.5px（全圆） | 1.5px（全圆） |
| 颜色 token | `color-red` | `color-divider` |
| 颜色值 | `#2E58FF` | Light `rgba(0,0,0,0.08)` / Dark `rgba(255,255,255,0.08)` |

**布局规则：**

- dot 间距（gap）：4px
- 指示器组整体**水平居中**于所在容器
- dot 数量 = 总页数；选中 dot 始终为最大宽度（12px）

---

### 02 自由滑动指示器

**适用场景：** 宫格、横向卡片列表等——滑动**不吸附**，停止位置跟随手指。

| 属性 | 规格 |
|---|---|
| 轨道宽度 | 24px（固定） |
| 轨道高度 | 3px |
| 圆角 | 1.5px（全圆） |
| 轨道颜色 token | `color-divider` |
| 轨道颜色值 | Light `rgba(0,0,0,0.08)` / Dark `rgba(255,255,255,0.08)` |
| 选中段颜色 token | `color-red` |
| 选中段颜色值 | `#2E58FF` |
| 选中段高度 | 3px |

**选中段宽度计算：**

```
选中段宽度 = round( 可见项数 / 总项数 × 24 )   // 四舍五入取整数，单位 px
```

**布局规则：**

- 整体轨道（24px）**水平居中**于所在容器
- 选中段始终从轨道左端起始，随滚动位置实时更新宽度

---

## 通用规格

| 属性 | Token | 值 |
|---|---|---|
| 高度 | — | 3px（固定原始值，无对应 sizing token） |
| 圆角 | — | 1.5px（= height/2，全圆；无对应 radius token） |
| 选中色 | `color-red` | `#2E58FF` |
| 未选中色 | `color-divider` | Light `rgba(0,0,0,0.08)` / Dark `rgba(255,255,255,0.08)` |

---

## 设计约束 (Do & Don't)

✅ 翻页指示器 dot 数量须与总页数严格对应  
✅ 自由滑动指示器选中段宽度按公式计算后四舍五入取整数  
✅ 两种类型指示器均须水平居中于容器  
❌ 两种类型不可混用——有吸附用翻页指示器，无吸附用自由滑动指示器  
❌ 选中颜色不可替换为其他颜色，固定使用 `#2E58FF`  
❌ 自由滑动指示器轨道总长度固定为 24px，不可根据屏幕宽度缩放
