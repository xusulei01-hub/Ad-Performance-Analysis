# Tab (选项卡)

## Overview

Tabs group content of the same level into switchable views. Choose the tab type based on **content hierarchy**, not visual preference.

---

## Type Hierarchy

Use the hierarchy level of the content to determine which tab type to use:


| Level | Component        | 中文   | 样式      | 高度                      | 字号          | When to Use             |
| ----- | ---------------- | ---- | ------- | ----------------------- | ----------- | ----------------------- |
| 1     | `Tab:01 页面一级tab` | 页面一级 | 下划线     | 40px（无副标题）/ 58px（有副标题）  | 16px        | 页面顶部主分类切换；可携带副标题        |
| 2     | `Tab:02 页面二级tab` | 页面二级 | 胶囊 pill | 46px（单行）/ 76px（双行）      | 14px        | 页面级内容分类；inactive 态灰底无边框 |
| 3     | `Tab:03 模块一级tab` | 模块一级 | 下划线     | 36px                    | 14px        | 模块内筛选；比 Tab:01 更紧凑      |
| 4     | `Tab:04 模块二级tab` | 模块二级 | 胶囊 pill | 46px（单行）/ 76px（双行）      | 14px        | 模块内二级筛选；inactive 态透明有描边 |
| 5     | `Tab:05 看板tab`   | 看板   | 数据卡片    | 三行约 90px / 两行 56px（单项高） | 14px + 18px | 每个 tab 项内嵌实时数据指标，用于看板场景 |
| 6     | `Tab:06 文字标签tab` | 文字标签 | 纯文字     | —                       | —           | 最弱层级；仅用于分时/F10 等特殊切换    |
| —     | `Tab:07 分段式tab`  | 分段式  | 分段控件    | 48px                    | 14px        | 功能上等同筛选/切换，不用于导航层级      |


> **Rule:** 先确定内容层级，再选对应 tab 类型。不要凭视觉感受选 tab 样式。

| 未选中 | Regular（400） | `rgba(0,0,0,0.60)`               |


各项之间以竖线分隔。

## Tab:01 页面一级 — 详细规范

Tab:01 是层级最高的 tab，用于页面顶部主分类切换。采用**下划线指示条**风格，且支持**副标题**模式。

### 变体矩阵

Tab:01 有两个正交维度，共 12 个变体：


| 分列 \ 副标题  | 无副标题          | 有副标题               |
| --------- | ------------- | ------------------ |
| `二等分`     | 2 tab 等分      | 2 tab 等分 + 副标题     |
| `三等分`     | 3 tab 等分      | 3 tab 等分 + 副标题     |
| `四等分`     | 4 tab 等分      | 4 tab 等分 + 副标题     |
| `五等分`     | 5 tab 等分      | 5 tab 等分 + 副标题     |
| `固定间距`    | 可滚动，64px/item | 可滚动 + 副标题，容器高 58px |
| `固定间距_编辑` | 可滚动 + 编辑图标    | 可滚动 + 副标题 + 编辑图标   |


### 容器尺寸


| 属性           | 无副标题                      | 有副标题                      |
| ------------ | ------------------------- | ------------------------- |
| 组件总高（等分变体）   | **40px**                  | 自适应（约 58px）               |
| 组件总高（固定间距变体） | **40px**                  | **58px**                  |
| 背景色          | `color-foreground-layer1` | `color-foreground-layer1` |


### 单个 Tab 项


| 属性          | 值         |
| ----------- | --------- |
| 上内边距        | 10px      |
| 下内边距        | 3px       |
| 宽度（等分变体）    | `flex: 1` |
| 宽度（固定间距变体）  | **64px**  |
| 固定间距 tab 间距 | **24px**  |


### 文字规范


| 元素      | 字号       | 行高   | 字重（active）  | 字重（inactive） | 颜色（active）                                | 颜色（inactive）                              |
| ------- | -------- | ---- | ----------- | ------------ | ----------------------------------------- | ----------------------------------------- |
| Tab 主标题 | **16px** | 20px | Medium（500） | Regular（400） | `color-text-primary` `rgba(0,0,0,0.84)`   | `color-text-secondary` `rgba(0,0,0,0.60)` |
| 副标题     | 11px     | 18px | Regular     | Regular      | `color-text-secondary` `rgba(0,0,0,0.60)` | `color-text-secondary` `rgba(0,0,0,0.60)` |


> 副标题在 active 和 inactive 状态下颜色相同，不随选中态变化。

### 选中指示条


| 属性   | 值                       |
| ---- | ----------------------- |
| 高度   | 3px                     |
| 宽度   | **12px**（固定，不随 tab 宽变化） |
| 水平位置 | 居中于 tab 项               |
| 圆角   | 1px                     |
| 颜色   | `color-red` `#2E58FF`   |
| 未选中态 | `opacity: 0`，位置保留       |


### 固定间距可滚动变体

- 右侧渐变遮罩：宽 24px，`color-foreground-layer1` → 透明
- `固定间距_编辑`：遮罩右侧叠加白色背景区域 + 16px 菜单图标（`图标/B45 菜单`）
- 有副标题时容器总高升至 **58px**，渐变遮罩高度同步

## Tab:02 页面二级 — 详细规范

Tab:02 采用**胶囊 pill 样式**，用于页面级内容分类（通常位于页面顶部）。与 Tab:04 外形相同，区别见下方对比表。

### 变体

`三等分` / `四等分` / `三等分-双行` / `四等分-双行` / `固定间距` / `固定间距_编辑`（共 6 个，结构与 Tab:04 一致）

### 容器尺寸


| 属性        | 值                                            |
| --------- | -------------------------------------------- |
| 组件总高（单行）  | **46px**（上下各 8px padding + 30px pill）        |
| 组件总高（双行）  | **76px**（上下各 8px padding + 30px×2 + 8px 行间距） |
| 上下内边距     | 8px                                          |
| tab 间水平间距 | 8px                                          |
| 背景色       | `color-foreground-layer1`                    |


### 单个 Tab 项（Pill）


| 属性       | 值             |
| -------- | ------------- |
| 高度       | **30px**      |
| 最小宽度     | **66px**      |
| 固定间距变体宽度 | **72px**      |
| 内边距      | 水平 8px，垂直 5px |
| 圆角       | **4px**       |
| 文字大小     | 14px，居中       |


### 视觉状态


| 状态            | 背景色                                           | 文字颜色                                      | 边框  | 字重              |
| ------------- | --------------------------------------------- | ----------------------------------------- | --- | --------------- |
| 选中（active）    | `color-transparent-red` `rgba(46,88,255,0.1)` | `color-red` `#2E58FF`                     | 无   | **Medium（500）** |
| 未选中（inactive） | `color-background-weak` `rgba(0,0,0,0.04)`    | `color-text-secondary` `rgba(0,0,0,0.60)` | 无   | Regular（400）    |


### 固定间距可滚动变体

- pill 固定 72px 宽，间距 8px，容器横向滚动
- 右侧渐变遮罩：白色 → 透明，宽 24px
- `固定间距_编辑`：遮罩右侧叠加白色背景 + 16px 菜单图标

### 双行布局

- 容器改为 `flex-col`，行间距 8px
- 第一行含选中高亮 pill，第二行为另一组 tab
- 每行内 tab 均等分（`flex: 1`），pill 高度固定 30px

## Tab:05 看板tab — 详细规范

Tab:05 是数据看板专用 tab。每个 tab 项不只是标签，而是内嵌实时行情数据的**指标卡片**，用于快速横向比较多个指标（如可转债指数、涨跌分布、均价等）。

### 变体矩阵

Tab:05 有两个正交维度，共 6 个变体：


| 行数 \ 分列                    | 二等分    | 三等分    | 固定间距    |
| -------------------------- | ------ | ------ | ------- |
| **三行**（title + 大数字 + 辅助信息） | 三行×二等分 | 三行×三等分 | 三行×固定间距 |
| **两行**（title + 辅助信息）       | 两行×二等分 | 两行×三等分 | 两行×固定间距 |


### 容器尺寸


| 属性        | 值                         |
| --------- | ------------------------- |
| 上内边距      | 12px                      |
| 下内边距      | 6px                       |
| tab 间水平间距 | 8px                       |
| 背景色       | `color-foreground-layer1` |


### 单个 Tab 项尺寸


| 行数变体      | 内容区高度       | 上内边距 | 下内边距 | 水平内边距 |
| --------- | ----------- | ---- | ---- | ----- |
| 三行（tall）  | 自适应（约 90px） | 12px | 18px | 4px   |
| 两行（short） | 固定 **56px** | 6px  | 12px | 12px  |


**宽度**：


| 分列变体      | 等分变体      | 固定间距变体                    |
| --------- | --------- | ------------------------- |
| 二等分 / 三等分 | `flex: 1` | —                         |
| 固定间距      | —         | 80px（两行）/ 100px（三行），可横向滚动 |


### 内容层级（三行变体）

```
[Tab 标题]      14px / font-family-ios-cn font-weight-medium / color-text-primary
[大数字]        18px / font-family-number Bold（THS JinRongTi Bold）/ 涨跌色
[辅助小信息]    11px / 混排（见下方数据类型）
```

### 内容层级（两行变体）

```

[辅助小信息]    11px / font-family-number Medium（THS JinRongTi Medium）/ 涨跌色
```

### 数据类型规范

#### 类型 A：点数（指数点位）


| 层   | 示例                | 字体                          | 字号   | 颜色                    |
| --- | ----------------- | --------------------------- | ---- | --------------------- |
| 大数字 | `1234.231`        | `font-family-number` Bold   | 18px | `color-price-up/down` |
| 辅助行 | `+207.160 +0.87%` | `font-family-number` Medium | 11px | `color-price-up/down` |


辅助行两段间距 6px，点位宽 48px + 涨幅宽 38px，总宽 92px 居中。

#### 类型 B：涨跌比（up:down 比例）


| 层    | 示例        | 字体                                         | 字号   | 颜色                             |
| ---- | --------- | ------------------------------------------ | ---- | ------------------------------ |
| 大数字  | `348：123` | `font-family-number` Bold                  | 18px | 涨色 + `rgba(0,0,0,0.4)` 冒号 + 跌色 |
| 辅助标签 | `正股`      | `font-family-ios-cn` `font-weight-regular` | 11px | `color-text-secondary`         |
| 辅助数据 | `234：25`  | `font-family-number` Medium                | 11px | 涨色 + 透明冒号 + 跌色                 |


辅助行两段间距 6px，总宽 92px 居中。

#### 类型 C：单一涨幅/均价


| 层    | 示例       | 字体                                         | 字号   | 颜色                     |
| ---- | -------- | ------------------------------------------ | ---- | ---------------------- |
| 大数字  | `5.31%`  | `font-family-number` Bold                  | 18px | `color-price-up/down`  |
| 辅助标签 | `溢价率`    | `font-family-ios-cn` `font-weight-regular` | 11px | `color-text-secondary` |
| 辅助数据 | `10.12%` | `font-family-number` Medium                | 11px | `color-text-primary`   |


辅助行两段间距 6px，总宽 92px 居中。

### 装饰背景

每个 tab 项内有抽象波浪形 **SVG Union 背景图**，绝对定位覆盖整个 tab 项区域。纯装饰，不影响布局，按实际 SVG 资源引用。

## Layout Rules

### 下划线型（Tab:01 · Tab:03）


| 规则            | 值                                     |
| ------------- | ------------------------------------- |
| 默认布局          | 等分（`flex: 1`），tabs 撑满容器宽度             |
| 切换为可滚动的条件     | item 数量 > 5，或等分后单项宽度过窄                |
| 固定间距模式 item 宽 | Tab:01 → 64px；Tab:03 → 56px，间距均为 24px |


### 胶囊型（Tab:02 · Tab:04）


| 规则            | 值                      |
| ------------- | ---------------------- |
| 默认布局          | 等分（`flex: 1`），pill 撑满列 |
| 最小 pill 宽度    | 66px                   |
| 切换为可滚动的条件     | item 数量较多，等分后宽度不足 66px |
| 固定间距模式 item 宽 | 72px，间距 8px            |
| 推荐 label 最大长度 | 4 个字符（2 个汉字 + 单位等）     |


### 分段控件（Tab:07）


| 规则          | 值                          |
| ----------- | -------------------------- |
| 最小 tab 宽度   | 66px                       |
| 内水平 padding | 12px（固定）                   |
| 最大列数        | 4 列（标准）；特殊场景可 3 列          |
| 圆角          | 内小外大（inner < outer），保持视觉协调 |

## Icon Support

By default, Tab components contain **text only**. The spec allows an optional icon before the label text (`支持tab前方添加图标`). If used, reference icons from `assets/icons/actions/` or `assets/icons/arrows/`.

## Constraints / Do & Don't


|     | Rule                                       |
| --- | ------------------------------------------ |
| ✅   | 先确定内容层级，再选对应 tab 类型                        |
| ✅   | 下划线型默认等分；仅当 item 数多或宽度不足时切换为固定间距可滚动        |

## Usage in Navigation Context

Tab:01 页面一级tab 通常紧贴导航栏下方：

- **一级导航栏 + Tab:01** → `导航栏/01一级导航栏/07资讯_tab_有编辑` / `08资讯_tab_无编辑`
- **标准导航栏 + Tab:01** → `导航栏/02标准导航栏/06返回+标题tab`

当 Tab 嵌入导航栏标题区（Tab in nav）时，使用导航栏组件的 `返回+标题tab` 变体，将 tab 保留在标准 44px 导航栏高度内，而不是叠放在导航栏下方。
